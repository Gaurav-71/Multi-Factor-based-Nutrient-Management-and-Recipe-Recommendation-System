# Backend

## Virtual Environemnt

- Python Version

  `3.9.4`

- Environement Name

  `venv`

- Install

  `python -m pip install virtualenv`

- Create

  `python -m venv ‘env-name’`

- Activate

  `source ‘env-name’/bin/activate`

- Exit

  `deactivate`

## Install Required Packages

`pip install -r requirements.txt`

## Flask setup

- Run Command

  `flask run`

- Setup environment variable (optional)

  `export FLASK_APP=main.py`
